how to create a profile README (takes 30 seconds, you have no excuse)

  1. Go to github.com/new
  2. Set the repository name to your exact GitHub username
  3. Check "Add a README file"
  4. Click "Create repository"
  5. Edit the README.md and paste the embed code above
  6. Commit — the chart now appears on your GitHub profile. you're welcome.
